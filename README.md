# Attendance System

A responsive web-based attendance management system with user authentication features.

## Features

- **User Registration**: Create new accounts with personal information
- **User Login**: Secure login with username and password
- **Password Reset**: Recover forgotten passwords via email
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Modern UI**: Clean, professional interface with intuitive navigation

## Project Structure

```
Attendance_System/
├── index.html          # Login page
├── register.html       # User registration page
├── reset.html          # Password reset page
├── assets/
│   └── logo.jpg        # Application logo
├── css/
│   └── index.css       # Main stylesheet
└── js/                 # JavaScript files (currently empty)
```

## Technologies Used

- **HTML5**: Semantic markup and form structure
- **CSS3**: Responsive design with Flexbox and media queries
- **JavaScript**: Client-side functionality (to be implemented)

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for better development experience)

### Installation

1. Clone or download the project files
2. Open the project folder in your code editor
3. Start a local web server or open `index.html` directly in your browser

### Usage

1. **Login**: Open `index.html` to access the login page
2. **Register**: Click "Don't have account? Register Now" to create a new account
3. **Reset Password**: Click "Forgot Password?" to reset your password

## Development

### File Organization

- **HTML Files**: Contain the page structure and forms
- **CSS**: `css/index.css` contains all styling including responsive breakpoints
- **Assets**: Store images and other media files in the `assets/` folder
- **JavaScript**: Add interactive functionality in the `js/` folder

### Responsive Breakpoints

- Desktop: Default styles
- Tablet: Max-width 860px
- Mobile: Max-width 520px
